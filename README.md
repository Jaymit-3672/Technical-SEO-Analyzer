# Technical SEO Analyzer

A Python-based Technical SEO Analyzer built using Flask and BeautifulSoup.

## Features

* SEO Score Calculation
* Title Tag Analysis
* Meta Description Analysis
* H1-H6 Heading Audit
* Image ALT Text Audit
* Sitemap.xml Detection
* Robots.txt Detection
* Canonical Tag Detection
* Internal Link Analysis
* External Link Analysis
* Unique Link Detection
* Schema Markup Detection
* Analysis Time Tracking

## Tech Stack

* Python
* Flask
* BeautifulSoup
* Requests
* HTML/CSS

## Installation

```bash
git clone https://github.com/Jaymit-3672/Technical-SEO-Analyzer.git

cd Technical-SEO-Analyzer

pip install -r requirements.txt

python app.py
```

## Usage

1. Run the application.
2. Open http://127.0.0.1:5000
3. Enter a website URL.
4. View the SEO audit report.
